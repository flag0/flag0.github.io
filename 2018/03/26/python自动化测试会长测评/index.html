0x04 分析url

经过不断的尝试可以发现

中的nquiae表示不同的问卷，每个社团的都不一样

而

中的openid和tocken可能代表每个填写答卷的用户id
分析发现openid参数的数值以obncMj开头的，长度为28的随机字符(也许是加密后的，我们这里用随机字符来模拟)，tocken是以4_开头的长度为109的随机字符(也许是加密后的，我们这里用随机字符来模拟)

构造randome函数来模拟(
random.sample会从字符串中随机取出指定长度的字符)

0x06姓名和留言
