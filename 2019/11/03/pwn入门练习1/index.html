gdb-peda常见命令

  • file path 打开文件
  • run & r 运行
  • step 步入执行
  • next 步过执行
  • disassemble+func 对制定的函数进行反汇编
  • break *0x400100 在0x400100处下断点
  • b mian 在mian函数处下断点
  • stack 20 查看堆栈
